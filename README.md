## How to run

- Go in the project folder
- Open a cmd
- Run 'pip instsall -r requirements.txt'
- Run 'uvicorn main:app'

After that, you'll see it running and you can visit the API at:

- http://127.0.0.1:8000/
- http://127.0.0.1:8000/docs#
- http://127.0.0.1:8000/redoc
